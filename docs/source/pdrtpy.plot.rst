Plotting
========

The :mod:`~pdrtpy.plot` module provides classes that are paired with tools in the :mod:`~pdrtpy.tool` module.   Currently, there are 2 classes :class:`~pdrtpy.plot.lineratioplot.LineRatioPlot` which is used to plot the results of :class:`~pdrtpy.tool.lineratiofit.LineRatioFit`, and :class:`~pdrtpy.plot.h2excitationplot.H2ExcitationPlot` that is used in :class:`~pdrtpy.tool.h2excitation.H2Excitation`.  All plots are derived from :class:`~pdrtpy.plot.plotbase.ToolBase`.
